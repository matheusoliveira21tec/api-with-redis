### 🔧 Installation

1. Clone the api-with-redis repository:
```sh
git clone https://github.com/matheusoliveira21tec/api-with-redis
```

2. Change to the project directory:
```sh
cd api-with-redis
```

3. Install the dependencies:
